Description

Celui-ci se veut comme étant une plateforme de blogue, sur laquelle il est possible d’effectuer toutes les interactions que ce type de plateforme sous-entend : créer un nouveau sujet, commenter, « like », etc. Comme ce programme est principalement destiné aux programmeurs de ce monde, il sera impossible pour le commun des mortels de l’utiliser immédiatement. Cependant, comme il est libre de droits, nous avons bon espoir qu’il sera intégré à de nombreux sites web et que vous pourrez l’utiliser un jour ou l’autre! En attendant, voici le lien pour avoir un aperçu du squelette de ce premier programme open source : http://blogproject.net/

Pour les programmeurs de cette planète, vous trouverez, en cliquant sur le lien ci-haut, la page GitHub du projet. Surtout, sentez-vous à l’aise et faites comme chez vous! Modifiez ce programme comme bon vous semble et intégrez-le à vos projets.

Pour ce qui est des technologies utilisées dans le projet, cette application utilise MongoDB comme système de base de données. Un Framework ExpressJS comme Back-End (Server). Et du côté Front-End (Navigateur Web), le Framework AngularJS nous permet d’utiliser la structure objet de la base de données combinée avec l’API en Back-End qui nous permet de récupérer ou modifier ces objets à l’aide de requête. Ces technologies ont pour but de viser la performance et la fluidité.
